Task 1 : API in NodeJS
Create an API in NodeJS that allows user login via username, password
a. username to be alphanumeric
b. username to be between 6-12 letters
c. password to allow alphabet, numbers, special characters
d. minimum password length is 6

*************Process to start the app************
1. First, go inside the Task-1 directory (cd Task-1), then go inside the server directory (cd Task-1/server)
2. Now when you reach inside the server, first run this command : "npm install", then run following command to start the backend "node index.js" or "nodemon index.js"
3. Now open another terminal and go to Task-1 directory, first run this command : "npm install", then run following command to start the frontend "npm run dev"
